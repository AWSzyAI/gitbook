# 一、读不懂就别读了：如何阅读？

与[Prindiple](https://szy-2.gitbook.io/szynote/)类似，阅读也分为两种性质：专业的、有用的、强目的性的阅读；与娱乐性的轻松阅读。

专业类的，包括：教材、文档、论文、博客。非专业类的有：知乎微博等文字平台、顶刊官网、邮箱SSR，重点考虑“文摘”甚至是“句摘”这种“碎片化阅读”。这类短句往往能够启发灵感，箴言类的可以促进人的自我审视甚至维护个人性格风格的一致性。
